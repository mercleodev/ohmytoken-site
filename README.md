# ohmytoken-site

Landing site for [OhMyToken](https://github.com/mercleodev/ohmytoken) — a privacy-first usage monitor for Claude, Codex, and Gemini.

## Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- Inter + JetBrains Mono via `next/font/google`

## Develop

```bash
npm install
npm run dev
```

Open <http://localhost:3000>.

## Build

```bash
npm run build
npm run start
```

## Deploy
